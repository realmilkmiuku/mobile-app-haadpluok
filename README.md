# Haadpluok - Home-grown Vegetable Mobile Application

## Overview
Sometimes, how to grow home-grown vegetables is difficult for beginners. This mobile application is intended to learn about growing home-grown vegetables and runs on the Android Operating System. Using XML and Kotlin languages with a purpose to gather home-grown vegetable knowledge foundations. It also facilitates and enhances planting management habits with users, and strengthens the discipline of caring by auto-generating activity functions or creating activity by user function.

## What I've learned
- Applied design thinking theory to design and implement features based on user requirements survey
- Designed UI mockups with Figma
- Developed mobile application features using Kotlin in Android Studio
- Implemented Firebase Authentication for register and login function
- Implemented CRUD functionalities
- Utilized Firebase Cloud Database for storing project data
- Integrated Facebook API for login and sharing content
